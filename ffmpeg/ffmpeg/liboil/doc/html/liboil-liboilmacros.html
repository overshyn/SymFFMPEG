oil_divide_255()

#define oil_divide_255(x) ((((x)+128) + (((x)+128)>>8))>>8)

Divides x by 255 in a way that is compatible with the pixel operations in Liboil. The number 65025 is 255*255.

Evaluates to x divided by 255.

x :
